# 实验室管理系统前端

## 安装与运行

### 前提条件
- Node.js 16+
- npm 或 yarn

### 安装依赖
```bash
cd frontend
npm install
# 或者如果使用 yarn
# yarn
```

### 开发模式运行
```bash
npm start
# 或者
# yarn start
```

### 构建生产版本
```bash
npm run build
# 或者
# yarn build
```

## 注意事项

- 不要提交 `node_modules` 目录到 Git 仓库
- 确保你的 `.gitignore` 文件正确配置
- 前端与后端的 API 交互默认指向 `http://127.0.0.1:8000`

## 技术栈

- React
- TypeScript
- Ant Design
- Axios
